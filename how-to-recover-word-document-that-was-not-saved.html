Can I retrieve a Word document that wasn’t saved?

Recovering Unsaved Word Documents

In MS Word, click the File Tab in the upper left. Click Manage Document and select Recover Unsaved Documents from the drop-down list. Check for your missing file in the dialog box. … Open the recovered Word document and click the Save As button in the top banner.

How do I recover a Word document that was not saved Windows 10?

Go to File / Manage Document and click on the File tab. Now select “Document Management”, then “Recover Unsaved Documents”. In the new window, you will see a list of all unsaved Word documents. Select the documents you are interested in and click Open.

Where are unsaved Word documents stored?

These can be found via File, Open and clicking the Recover Unsaved Documents button found at the very bottom of the Recent File List. Open Word and select File, Options. In the Options dialog box select Save from the left hand menu. Note the AutoRecover files location.

Does Word keep backup files?

Microsoft Word allows you to make an automatic backup of your working document. If the program crashes or your computer accidentally loses power, the AutoRecover feature will attempt to load the most up-to-date version of your document.

How long are unsaved Word documents kept?

    What causes files to disappear?

    Files can disappear when the properties are set to “hidden” and File Explorer is not configured to show hidden files. Computer users, programs and malware can edit file properties and set them to hidden to give the illusion that the files don’t exist and prevent you from editing the files.

    Why do my Word documents disappear?

    Microsoft Word document can disappear in certain situations. For example, it may be gone after a forced quit, power interruption or close without saving changes.
